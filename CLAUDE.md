# sapling

sapling is an embedded data storage solution for pydantic models, built on
sqlite. it strives to be very simple

## dev workflow

- see [README.md](./README.md) for dev workflow including running tests
- reference docs linked in the README
  for a deeper understanding of the inspiration for this project, and its dependencies
- after any changes, run linter, formatter and type checker
- run tests to validate your changes
- DO NOT ignore lint errors in lines or type errors.
  if you get stumped, pause and ask how to proceed
